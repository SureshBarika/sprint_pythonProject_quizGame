Q: What does HTML stand for?
A. Hyper Text Markdown Language
B. Hyper Text Markup Language
C. Home Tool Markup Language
D. Hyperlinks and Text Markup Language
Answer: B

Q: Which HTML tag is used to define an internal style sheet?
A. <css>
B. <script>
C. <style>
D. <link>
Answer: C

Q: What is the correct syntax for referencing an external script called "app.js"?
A. <script href="app.js">
B. <script src="app.js">
C. <script ref="app.js">
D. <script link="app.js">
Answer: B

Q: In CSS, what does 'position: absolute' do?
A. Positions the element relative to the page
B. Positions the element relative to its closest positioned ancestor
C. Fixes the element at the top
D. Makes the element scroll with the page
Answer: B

Q: Which React hook is used to manage state in functional components?

Q: Which property is used in CSS to change the text color?
